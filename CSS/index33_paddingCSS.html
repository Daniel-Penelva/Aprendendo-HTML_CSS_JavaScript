CSS Padding (Preenchimento)

O preenchimento é usado para criar espaço ao redor do conteúdo de um elemento, dentro de qualquer borda definida.

Com CSS, você tem controle total sobre o preenchimento. Existem propriedades para definir o preenchimento de cada lado de um elemento(top, right, bottom, and left).

Este elemento tem um padding de 70px.

-----------------------------------------------------------------------------------

Preenchimento - Lados Individuais

CSS tem propriedades para especificar o preenchimento de cada lado de um elemento:

padding-top

padding-right

padding-bottom

padding-left

Todas as propriedades de preenchimento podem ter os seguintes valores:

length - especifica um preenchimento em px, pt, cm, etc.

% - especifica um preenchimento em % da largura do elemento que o contém.

inherit- especifica que o preenchimento deve ser herdado do elemento pai.

Nota: Valores negativos não são permitidos.

Usando propriedades de padding individuais

Este elemento div tem um preenchimento superior de 50px, um preenchimento direito de 30px, um preenchimento inferior de 50px e um preenchimento esquerdo de 80px.

-----------------------------------------------------------------------------------

Preenchimento - propriedade abreviada

Para encurtar o código, é possível especificar todas as propriedades de preenchimento em uma propriedade.

padding propriedade é uma propriedade abreviada para as seguintes propriedades de preenchimento individuais:

Se a padding tiver quatro valores:

padding: 25px 50px 75px 100px;

top padding é 25px

right padding é 50px

bottom padding é 75px

left padding é 100px

A propriedade abreviada de preenchimento - 4 valores

Este elemento div tem um preenchimento superior de 25px, um preenchimento direito de 50px, um preenchimento inferior de 75px e um preenchimento esquerdo de 100px.

-----------------------------------------------------------------------------------

Se a padding tiver três valores:

padding: 25px 50px 75px;

padding-top é 25px

padding right e padding-left são 50px

padding-bottom é 75px

A propriedade abreviada de preenchimento - 3 valores

Este elemento div tem um preenchimento superior de 25px, um preenchimento direito e esquerdo de 50px e um preenchimento inferior de 75px.

-----------------------------------------------------------------------------------

Se a padding tiver dois valores:

padding: 25px 50px;

padding-top e padding-bottom é 25px

padding right e padding-left são 50px

A propriedade abreviada de preenchimento - 2 valores

Este elemento div tem um preenchimento superior e inferior de 25px e um preenchimento direito e esquerdo de 50px.

-----------------------------------------------------------------------------------

Se a padding tiver um valor:

padding: 25px;

A propriedade abreviada de preenchimento - 1 valor

Este elemento div tem um preenchimento superior, inferior, direito e esquerdo de 25px.

-----------------------------------------------------------------------------------

Preenchimento e largura do elemento

A propriedade CSS widthespecifica a largura da área de conteúdo do elemento. A área de conteúdo é a parte dentro do preenchimento, borda e margem de um elemento ( o modelo de caixa ).

Portanto, se um elemento tiver uma largura especificada, o preenchimento adicionado a esse elemento será adicionado à largura total do elemento. Isso geralmente é um resultado indesejável.

Preenchimento e largura do elemento

Este div tem 300px de largura.

A largura deste div é 350px, embora seja definido como 300px no CSS.

Para manter a width em 300px, independentemente da quantidade de preenchimento, você pode usar a box-sizing propriedade. Isso faz com que o elemento mantenha sua largura real; se você aumentar o preenchimento, o espaço de conteúdo disponível diminuirá.

Preenchimento e largura do elemento - com tamanho de caixa (box-sizing)

Este div tem 300px de largura.

A largura deste div permanece em 300px, apesar dos 50px de preenchimento total à esquerda e à direita, por causa da propriedade box-sizing: border-box.
