A propriedade de fundo

A propriedade background é uma propriedade abreviada para especificar todos os propriedades de plano de fundo em uma declaração.

Aqui, a imagem de fundo é mostrada apenas uma vez e também é posicionada no canto superior direito.

Também adicionamos uma margem direita, para que o texto não seja sobrescrito a imagem de fundo.
