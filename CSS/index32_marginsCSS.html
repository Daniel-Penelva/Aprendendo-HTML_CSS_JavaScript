Margens CSS

As marginpropriedades CSS são usadas para criar espaço ao redor dos elementos, fora de qualquer borda definida.

Com CSS, você tem controle total sobre as margens. Existem propriedades para definir a margem para cada lado de um elemento (top, right, bottom, and left).

Margem - Lados Individuais

CSS tem propriedades para especificar a margem para cada lado de um elemento:

margem - Uma propriedade abreviada para definir todas as propriedades de margem em uma declaração.

margin-top - Define a margem superior de um elemento.

margin-right - Define a margem direita de um elemento.

margin-bottom - Define a margem inferior de um elemento.

margin-left - Define a margem esquerda de um elemento.

Todas as propriedades de margem podem ter os seguintes valores:

auto - o navegador calcula a margem

length (comprimento) - especifica uma margem em px, pt, cm, etc.

% - especifica uma margem em % da largura do elemento recipiente

inherit - especifica que a margem deve ser herdada do elemento pai

Dica: Valores negativos são permitidos.

Usando propriedades de margem individuais

Este elemento div tem uma margem superior de 100px, uma margem direita de 150px, uma margem inferior de 100px e uma margem esquerda de 80px.

Margens Propriedade Abreviada

Para encurtar o código, é possível especificar todas as propriedades de margem em uma propriedade.

A margin propriedade é uma propriedade abreviada para as seguintes propriedades de margem individuais:

margin-top

margin-right

margin-bottom

margin-left

Então, aqui está como funciona:

Se a margin propriedade tiver quatro valores:

margem: 25px 50px 75px 100px = margin(top, right, bottom, left);

margem superior é 25px

margem direita é 50px

margem inferior é 75px

margem esquerda é 100px

A propriedade abreviada de margem - 4 valores

Este elemento div tem uma margem superior de 25px, uma margem direita de 50px, uma margem inferior de 75px e uma margem esquerda de 100px.

-----------------------------------------------------------------------------------

Se a margin propriedade tiver três valores:

margem: 25px 50px 75px = margin(top, right e left, bottom);

margem superior é 25px

margem direita e esquerda são 50px

margem inferior é 75px

A propriedade abreviada de margem - 3 valores

Este elemento div tem uma margem superior de 25px, uma margem direita e esquerda de 50px e uma margem inferior de 75px.

-----------------------------------------------------------------------------------

Se a margin propriedade tiver dois valores:

margem: 25px 50px = margin(top e bottom, right e left);

margem superior e inferior é 25px

margem direita e esquerda são 50px

A propriedade abreviada de margem - 2 valores

Este elemento div tem uma margem superior e inferior de 25px e uma margem direita e esquerda de 50px.

-----------------------------------------------------------------------------------

Se a margin propriedade tiver um valor:

margem: 25px

todas as quatro margens são 25px

A propriedade abreviada de margem com um valor

Este elemento div tem margens superior, inferior, esquerda e direita de 25px.

-----------------------------------------------------------------------------------

O valor automático

Você pode definir a propriedade margin para ''auto' para horizontalmente o elemento em seu contêiner.

O elemento ocupará a largura especificada e o espaço restante será dividido igualmente entre as margens esquerda e direita.

Este div será centralizado horizontalmente porque possui margem: auto;

-----------------------------------------------------------------------------------

O valor herdado

Este exemplo permite que a margem esquerda do elemento "p class='ex1'"" seja herdada do elemento pai div.

This paragraph has an inherited left margin (from the div element).

-----------------------------------------------------------------------------------

Colapso de Margin

Às vezes, as margens superior e inferior dos elementos são recolhidas em uma única margem que é igual à maior das duas margens.

Isso não acontece nas margens esquerda e direita! Apenas as margens superior e inferior!

Neste exemplo, o elemento h1 tem uma margem inferior de 50px e o elemento h2 tem uma margem superior de 20px. Portanto, a margem vertical entre h1 e h2 deveria ser de 70px (50px + 20px). No entanto, devido ao colapso da margem, a margem real acaba sendo 50px.
