Borda CSS - Lados Individuais

No CSS, também existem propriedades para especificar cada uma das bordas (top, right, bottom, and left).

Outra forma de ser feito

2 estilos de borda diferentes.

Então, aqui está como funciona

Se a border-style propriedade tiver quatro valores:

Estilo de borda: pontilhado sólido duplo tracejado

borda superior é pontilhada = top border is dotted

borda direita é sólida = right border is solid

borda inferior é dupla = bottom border is double

borda esquerda é tracejada = left border is dashed

Se a border-style propriedade tiver três valores:

estilo de borda: duplo sólido pontilhado

borda superior é pontilhada = top border is dotted

as bordas direita e esquerda são sólidas = right and left borders are solid

borda inferior é dupla = bottom border is double

Se a border-style propriedade tiver dois valores:

estilo de borda: sólido pontilhado

as bordas superior e inferior são pontilhadas = top and bottom borders are dotted

as bordas direita e esquerda são sólidas = right and left borders are solid

Se a border-style propriedade tiver um valor:

estilo de borda: pontilhado

todas as quatro bordas são pontilhadas = all four borders are dotted
