CSS Outline (contorno)

Este elemento tem uma borda preta de 2px e um contorno verde com uma largura de 10px.

---------- Explicação ----------

Outline CSS

Um contorno é uma linha desenhada ao redor dos elementos, FORA das bordas, para fazer o elemento "se destacar".

CSS tem as seguintes propriedades de contorno:

---------- outline-style (estilo) ----------

A outline-style propriedade especifica o estilo do contorno e pode ter um dos seguintes valores:

dotted- Define um contorno pontilhado

dashed- Define um contorno tracejado

solid- Define um contorno sólido

double- Define um contorno duplo

groove- Define um contorno ranhurado 3D

ridge- Define um contorno estriado 3D

inset- Define um contorno de inserção 3D

outset- Define um esboço inicial 3D

none- Não define contorno

hidden- Define um contorno oculto

A propriedade outline-style

Um contorno pontilhado

Um contorno tracejado

Um contorno sólido

Um contorno duplo

Um contorno de ranhura. O efeito depende do valor da cor do contorno.

Um contorno de crista. O efeito depende do valor da cor do contorno.

Um contorno inserido. O efeito depende do valor da cor do contorno.

Um esboço inicial. O efeito depende do valor da cor do contorno.

---------- outline-color (cor) ----------

A outline-color propriedade é usada para definir a cor do contorno.

A cor pode ser definida por:

nome - especifique um nome de cor, como "vermelho"

HEX - especifique um valor hexadecimal, como "#ff0000"

RGB - especifique um valor RGB, como "rgb(255,0,0)"

HSL - especifique um valor HSL, como "hsl(0, 100%, 50%)"

inverter - realiza uma inversão de cores (o que garante que o contorno fique visível, independentemente da cor do fundo)

A propriedade outline-color é usada para definir a cor do contorno.

Um contorno vermelho sólido.

Um contorno azul pontilhado.

Um contorno cinza inicial.

A outline-width propriedade especifica a largura do contorno e pode ter um dos seguintes valores:

---------- outline off-set (deslocamento) ----------

A outline-color propriedade é usada para definir a cor do contorno.

A propriedade de deslocamento (off-set) de contorno adiciona espaço entre um contorno e a aresta/borda de um elemento.

O espaço entre um elemento e seu contorno é transparente.

---------- outline (abreviado) ----------

A outlinepropriedade é uma propriedade abreviada para definir as seguintes propriedades de contorno individuais:

outline-width

outline-style(requeridos)

outline-color

A outline propriedade é especificada como um, dois ou três valores da lista acima. A ordem dos valores não importa.
