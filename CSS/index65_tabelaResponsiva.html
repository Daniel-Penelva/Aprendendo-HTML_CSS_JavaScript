Tabela responsiva

Uma tabela responsiva exibirá uma barra de rolagem horizontal se a tela estiver muito pequeno para exibir o conteúdo completo. Redimensione a janela do navegador para ver o efeito:

Para criar uma tabela responsiva, adicione um elemento container (como div) com 'overflow-x:auto' ao redor do elemento da tabela:
