Layout CSS - display para visibilidade

A displaypropriedade é a propriedade CSS mais importante para controlar o layout.

A displaypropriedade especifica se/como um elemento é exibido.

Cada elemento HTML tem um valor de exibição padrão, dependendo do tipo de elemento. O valor de exibição padrão para a maioria dos elementos é:

Inline - visibilidade na mesma linha

Block - visibilidade em bloco

none - visibilidade oculta
