Igual ao nome da cor "Tomate":

rgb(255, 99, 71)

#ff6347

hsl(9, 100%, 64%)

Igual ao nome da cor "Tomate", mas 50% transparente:

rgba(255, 99, 71, 0.5)

Além dos nomes de cores predefinidos, as cores podem ser especificadas usando RGB, HEX, HSL ou até cores transparentes usando valores de cores RGBA ou HSLA.

------ Explicação ------

Valores de Cores CSS

Valor RGB

Em CSS, as cores também podem ser especificadas usando valores RGB - ex: RGB(255,99,71);

Em CSS, uma cor pode ser especificada como um valor RGB, usando esta fórmula:

rgb ( vermelho, verde , azul )

Cada parâmetro (vermelho, verde e azul) define a intensidade da cor entre 0 e 255.

Para exibir branco, defina todos os parâmetros de cor para 255, assim: rgb(255, 255, 255).

rgb(255, 0, 0)

rgb(0, 0, 255)

rgb(60, 179, 113)

rgb(238, 130, 238)

rgb(255, 165, 0)

rgb(106, 90, 205)

Os tons de cinza geralmente são definidos usando valores iguais para todas as 3 fontes de luz:

rgb(60, 60, 60)

rgb(90, 90, 90)

rgb(120, 120, 120)

rgb(180, 180, 180)

rgb(210, 210, 210)

rgb(240, 240, 240)

------------------------------------------------------------------------

Valor REX

Em CSS, as cores também podem ser especificadas usando valores HEX - ex: #ff6347.

Uma cor hexadecimal é especificada com: #RRGGBB, onde os inteiros hexadecimais RR (vermelho), GG (verde) e BB (azul) especificam os componentes da cor.

Em CSS, uma cor pode ser especificada usando um valor hexadecimal na forma:

#rrggbb _

Onde rr (vermelho), gg (verde) e bb (azul) são valores hexadecimais entre 00 e ff (o mesmo que decimal 0-255).

Para exibir preto, defina todos os valores como 00, assim: #000000.

Para exibir branco, defina todos os valores para ff, assim: #ffffff.

#ff0000

#0000ff

#3cb371

#ee82ee

#ffa500

#6a5acd

Os tons de cinza geralmente são definidos usando valores iguais para todas as 3 fontes de luz:

#3c3c3c

#616161

#787878

#b4b4b4

#f0f0f0

#f9f9f9

------------------------------------------------------------------------

Valor HSL

Em CSS, as cores também podem ser especificadas usando valores HSL - ex: hsl(9,100%,64%).

HSL significa matiz, saturação e leveza.

Em CSS, uma cor pode ser especificada usando matiz, saturação e luminosidade (HSL) na forma:

hsl( matiz , saturação , luminosidade )

Matiz é um grau na roda de cores de 0 a 360. 0 é vermelho, 120 é verde e 240 é azul.

A saturação é um valor percentual. 0% significa um tom de cinza e 100% é a cor total.

A leveza também é uma porcentagem. 0% é preto, 50% não é nem claro nem escuro, 100% é branco.

A saturação pode ser descrita como a intensidade de uma cor.

100% é cor pura, sem tons de cinza.

50% é 50% cinza, mas você ainda pode ver a cor.

0% é totalmente cinza; você não pode mais ver a cor.

hsl(0, 100%, 50%)

hsl(0, 80%, 50%)

hsl(0, 60%, 50%)

hsl(0, 40%, 50%)

hsl(0, 20%, 50%)

hsl(0, 0%, 50%)

Leveza - A claridade de uma cor pode ser descrita como quanta luz você deseja dar à cor, onde 0% significa ausência de luz (preto), 50% significa 50% de luz (nem escuro nem claro) e 100% significa total luminosidade (branco).

hsl(0, 100%, 0%)

hsl(0, 100%, 25%)

hsl(0, 100%, 50%)

hsl(0, 100%, 75%)

hsl(0, 100%, 90%)

hsl(0, 100%, 100%)

Tons de Cinza - Os tons de cinza geralmente são definidos definindo o matiz e a saturação como 0 e ajustando a luminosidade de 0% a 100% para obter tons mais escuros/claros:

hsl(0, 0%, 0%)

hsl(0, 0%, 24%)

hsl(0, 0%, 47%)

hsl(0, 0%, 71%)

hsl(0, 0%, 94%)

hsl(0, 0%, 100%)

------------------------------------------------------------------------

Valor RGBA

Em CSS, as cores também podem ser especificadas usando valores RGBA - ex: rgba(255,99,71,0.5); É um valor mais 50% transparente.

Os valores de cores RGBA são uma extensão dos valores de cores RGB com um canal alfa - que especifica a opacidade de uma cor.

Um valor de cor RGBA é especificado com:

rgba( vermelho, verde , azul, alfa )

O parâmetro alfa é um número entre 0,0 (totalmente transparente) e 1,0 (nada transparente):

rgba(255, 99, 71, 0)

rgba(255, 99, 71, 0.2)

rgba(255, 99, 71, 0.4)

rgba(255, 99, 71, 0.6)

rgba(255, 99, 71, 0.8)

rgba(255, 99, 71, 1)

------------------------------------------------------------------------

Valor HSLA

Em CSS, as cores também podem ser especificadas usando valores HSLA - ex: hsla(9,100%,64%,0.5). É um valor mais 50% transparente.

Os valores de cores HSLA são uma extensão dos valores de cores HSL com um canal alfa - que especifica a opacidade de uma cor.

Um valor de cor HSLA é especificado com:

hsla( matiz, saturação , luminosidade, alfa )

O parâmetro alfa é um número entre 0,0 (totalmente transparente) e 1,0 (nada transparente)

hsla(9, 100%, 64%, 0)

hsla(9, 100%, 64%, 0.2)

hsla(9, 100%, 64%, 0.4)

hsla(9, 100%, 64%, 0.6)

hsla(9, 100%, 64%, 0.8)

hsla(9, 100%, 64%, 1)
