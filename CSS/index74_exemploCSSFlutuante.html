Exemplo 3

Caixas de altura igual

Caixas flutuantes com alturas iguais:

Caixa 1

Some content, some content, some content

Caixa 2

Algum conteúdo, algum conteúdo, algum conteúdo

Algum conteúdo, algum conteúdo, algum conteúdo

Algum conteúdo, algum conteúdo, algum conteúdo

Este exemplo não é muito flexível. Não há problema em usar a altura do CSS se você puder garantir que as caixas sempre terão a mesma quantidade de conteúdo, mas nem sempre é o caso. Se você tentar o exemplo acima em um celular (ou redimensionar a janela do navegador), verá que o conteúdo da segunda caixa será exibido fora da caixa.

Volte ao tutorial e encontre outra solução, se não for o que você deseja.
