Código do computador

Algum código de programação:

x = 5; y = 6; z = x + y;

Observe que o code elemento não preserva espaços em branco extras e quebras de linha.

Para corrigir isso, você pode colocar o code elemento dentro de um (pre) elemento: 

        
        x = 5;
        y = 6;
        z = x + y;
