-------------------- Explicação --------------------

O atributo href

A 'a' tag define um hiperlink. O href atributo especifica a URL da página para a qual o link vai.

-------------------- Bom Saber --------------------

O atributo de destino

Por padrão, a página vinculada será exibida na janela atual do navegador. Para alterar isso, você deve especificar outro destino para o link.

O target atributo especifica onde abrir o documento vinculado.

O targetatributo pode ter um dos seguintes valores:

_self - Padrão. Abre o documento na mesma janela/guia em que foi clicado

_blank - Abre o documento em uma nova janela ou guia

_parent - Abre o documento no quadro pai

_top - Abre o documento em todo o corpo da janela

O atributo alvo

Visite W3Schools!

Se target="_blank", o link será aberto em uma nova janela ou guia do navegador.

Marcadores de links

Os marcadores podem ser úteis se uma página da Web for muito longa.

Para criar um marcador - primeiro crie o marcador e, em seguida, adicione um link a ele.

Quando o link for clicado, a página rola para baixo ou para cima até o local com o marcador.
