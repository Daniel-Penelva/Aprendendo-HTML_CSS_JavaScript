Imagem de fundo

Uma imagem de fundo para um elemento p:

Você pode especificar imagens de plano de fundo
para qualquer elemento HTML visível.
Neste exemplo, a imagem de fundo
é especificado para um elemento p.
Por padrão, a imagem de fundo
se repetirá na(s) direção(ões)
onde é menor que o elemento
onde é especificado. (Tente redimensionar o
janela do navegador para ver como o
imagem de fundo se comporta.
