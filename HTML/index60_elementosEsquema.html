Elementos de Esquema HTML

HTML tem vários elementos semânticos que definem as diferentes partes de uma página web:

header - Define um cabeçalho para um documento ou uma seção

nav - Define um conjunto de links de navegação

section - Define uma seção em um documento

article - Define um conteúdo independente e independente

aside - Define o conteúdo além do conteúdo (como uma barra lateral)

footer - Define um rodapé para um documento ou uma seção

details - Define detalhes adicionais que o usuário pode abrir e fechar sob demanda

summary - Define um título para o details elemento

Londres

Londres é a capital da Inglaterra. É a cidade mais populosa do o Reino Unido, com uma área metropolitana de mais de 13 milhões habitantes.

Situada às margens do rio Tâmisa, Londres tem sido um importante assentamento para dois milênios, cuja história remonta à sua fundação pelos romanos, que o batizou de Londinium.
