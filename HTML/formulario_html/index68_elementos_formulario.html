O elemento input e label




O Elemento select

O elemento select define uma lista suspensa:

Opção Pré-selecionada (selected)

Você pode pré-selecionar uma opção com o atributo selecionado:

Valores de Opções Visíveis (size)

Use o atributo size para especificar o número de valores visíveis.



Permitir Seleções Múltiplas (multiple)

Use o atributo multiple para permitir que o usuário selecione mais de um valor.



Pressione a tecla Ctrl (Windows) / Command (Mac) para selecionar várias opções.

Textarea

O elemento textarea define um campo de entrada multilinha.
