O atributo de valor de entrada

O atributo de valor especifica um valor inicial para um campo de entrada:






Atributo somente leitura de entrada

O atributo somente leitura especifica que um campo de entrada deve ser somente leitura (não pode ser alterado):






Atributo de entrada desabilitado

O atributo disabled especifica que um campo de entrada deve ser desabilitado (inutilizável e não clicável):






Atributo de tamanho de entrada

O atributo de tamanho especifica a largura (em caracteres) de um campo de entrada:






Atributo maxlength de entrada

O atributo maxlength especifica o número máximo de caracteres permitidos em um campo de entrada:






Atributos de entrada min e max

Os atributos min e max especificam os valores mínimo e máximo para um elemento de entrada.







Atributos de entrada múltipla

O atributo múltiplo especifica que o usuário tem permissão para inserir mais de um valor em um campo de entrada.



Para selecionar vários arquivos, mantenha pressionada a tecla CTRL ou SHIFT enquanto seleciona.

O atributo placeholder de entrada

O atributo placeholder especifica uma breve dica que descreve o valor esperado de um campo de entrada.



Atributo obrigatório de entrada

O atributo obrigatório especifica que um campo de entrada deve ser preenchido antes do envio do formulário.

Atributo de entrada step

O atributo step especifica os intervalos numéricos legais para um elemento de entrada.

Atributo de foco automático de entrada

O atributo de foco automático especifica que o campo de entrada deve receber foco automaticamente quando a página for carregada.






Atributos de altura e largura de entrada

Os atributos de altura e largura especificam a altura e a largura de um elemento de entrada type="image".





Observação: O tipo de entrada type="image" envia as coordenadas X e Y do clique que ativou o botão de imagem.

Atributo de lista de entrada

O atributo de lista refere-se a um elemento datalist que contém opções predefinidas para um elemento de entrada.

O atributo autocomplete

O atributo autocomplete especifica se um campo de entrada deve ou não ter o preenchimento automático habilitado.

Preencha e envie o formulário e, em seguida, recarregue a página para ver como o preenchimento automático funciona.

Observe que o preenchimento automático está "ativado" para o formulário, mas "desativado" para o campo de e-mail!
