Tamanho de Tabelas

As tabelas HTML podem ter tamanhos diferentes para cada coluna, linha ou tabela inteira.

Use o style atributo com as propriedades width (largura) ou height (altura) para especificar o tamanho de uma tabela, linha ou coluna.
