Espaçamento de Células (border-spacing)

Espaçamento celular é o espaço entre cada célula.

Por padrão, o espaço é definido como 2 pixels.

Para alterar o espaço entre as células da tabela, use a border-spacingpropriedade CSS no table elemento

Altere o espaço entre as células com a propriedade border-spacing.
