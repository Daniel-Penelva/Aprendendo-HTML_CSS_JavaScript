The picture Element

Redimensionar o navegador para ver diferentes versões do carregamento da imagem em diferentes tamanhos de viewport. O navegador procura o primeiro elemento de origem em que a consulta de média corresponda à largura atual da janela de visualização do usuário, e busca a imagem especificada no atributo srcset.

O elemento img é necessário como a última tag filho do bloco de declaração de imagem. O elemento img é usado para fornecer compatibilidade com versões anteriores para navegadores que não suportam o elemento picture ou se nenhuma das tags de origem corresponder.

Observação: o navegador usará o primeiro elemento com valores de atributo correspondentes e ignorar� os 'source' elementos seguintes.

Observação: o elemento de imagem não é suportado no IE12 e anterior ou no Safari 9.0 e anterior.

Bom saber

O elemento HTML 'picture' oferece aos desenvolvedores da Web mais flexibilidade na especificação de recursos de imagem.

O 'picture' elemento contêm um ou mais 'source' elementos, cada um referindo-se a diferentes imagens por meio do srcset atributo. Desta forma, o navegador pode escolher a imagem que melhor se adapta à visualização e/ou dispositivo atual.

Cada 'source' elemento possui um mediaatributo que define quando a imagem é a mais adequada.
