Listas HTML

As listas HTML permitem que os desenvolvedores da Web agrupem um conjunto de itens relacionados em listas.

Uma lista HTML não ordenada

Uma lista não ordenada começa com a 'ul' tag. Cada item da lista começa com a 'li' tag.

Os itens da lista serão marcados com marcadores (pequenos círculos pretos) por padrão

  • Café
  • Chá
  • Leite

Escolha de marcadores de lista NÃO ordenado

A propriedade CSS list-style-typeé usada para definir o estilo do marcador de item da lista. Pode ter um dos seguintes valores:

disc = marcador (padrão)

circle = círculo

square = quadrado

none = não será marcado

Uma lista HTML ordenada

Uma lista ordenada começa com a 'ol' tag. Cada item da lista começa com a 'li' tag.

Os itens da lista serão marcados com números por padrão

  1. Café
  2. Chá
  3. Leite

Escolha de marcadores de lista ordenado

O type atributo da 'ol' tag, define o tipo de marcador do item da lista:

type="1" - numerados com números

type="A" - letras maiúsculas

type="a" - letras minúsculas

type="I" - números romanos maiúsculos

type="i" - número romanos minúsculos

Contagem da lista de controle

Por padrão, uma lista ordenada começará a contar a partir de 1. Se você quiser começar a contar a partir de um número especificado, poderá usar o start atributo:

  1. Café
  2. Chá
  3. Leite
  1. Café
  2. Chá
  3. Leite

Listas aninhadas

As listas podem ser aninhadas (lista dentro da lista):

  1. Café
  2. Chá
    1. Chá preto
    2. Chá verde
  3. Leite

Lista de Descrição

Uma lista de descrição é uma lista de termos, com uma descrição de cada termo.

A 'dl' tag define a lista de descrição, a 'dt' tag define o termo (nome) e a 'dd' tag descreve cada termo:

Café
- bebida quente preta
Leite
- bebida gelada branca
